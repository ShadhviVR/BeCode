10.0 exemple d'asynchronicité

La notion d'asynchronicité en JavaScript est un concept incontournable malheureusement assez mal compris par les débutants.

On peut raisonnablement avancer qu'en comprendre les principes est un jalon important de l'apprentissage de JavaScript.
